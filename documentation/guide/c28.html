1.1. What is Cal3D?

Cal3D is a free skeletal-based 3d character animation library written in C++ in a platform/graphic-API independent way.

Originally designed to be used in a 3d client for WorldForge, it evolved into a stand-alone library which can be used in many different kinds of projects.
